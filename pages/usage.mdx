## Using ContentHook in the Frontend
In this section we will provide you with some information about using ContentHook in the Frontend and a example code.

### Configuration
- for configuration, please go to the [Configuration](/configuration)

### Choosing a Framework, Library or Tool
- ContentHook is basically compatible with all frameworks, libraries and tools as it is a CLI tool and you only Import the content files via an NPM package. You can use it with `React`, `Vue`, `Angular`, `Svelte`, `etc.`

I have choosen `React` for this example. Below are 2 examples, the first one is a new page function showing all available contents in form as blog posts, the second one is a single page function showing a single content of a blog post.

### Example Code
```jsx copy
import React, { useEffect, useState } from "react";
import { getContents } from "@contenthook/browser";

const Blog = () => {
    const [contents, setContents] = useState([]);

    useEffect(() => {
        const fetchContents = async () => {
            const contents = await getContents({ api_key: "YOUR_API_KEY" });
            setContents(contents);
        };

        fetchContents();
    }, []);

    return (
        <div>
            {contents.map((content, index) => (
                <div key={index}>
                    <h1>{content.metadata.title}</h1>
                    <p>{content.metadata.description}</p>
                </div>
            ))}
        </div>
    );
};

export default Blog;
```

```jsx copy
import React, { useEffect, useState } from "react";
import { getContents } from "@contenthook/browser";

const BlogPost = () => {
    const [content, setContent] = useState({});

    useEffect(() => {
        const fetchContent = async () => {
            const content = await getContents({ api_key: "YOUR_API_KEY" });
            setContent(content[0]); // As we only use one content in this example.
            // You can also use special ID's or slugs to get a specific content.
        };

        fetchContent();
    }, []);

    return (
        <div>
            <h1>{content.metadata.title}</h1>
            <p>{content.metadata.description}</p>
        </div>
    );
};

export default BlogPost;
```
