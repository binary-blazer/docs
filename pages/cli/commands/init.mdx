# Init

To initialize a new project, run the following command:
```bash copy
$ contenthook init
```

This will create a new file, `contenthook.config.js`, in the root of your project. This file will contain the default configuration for your project.
if clauses:
- If selected, It also can create a .env file for the `API_KEY` variable.
- If not yet created, it will create a `package.json` file for the project.
