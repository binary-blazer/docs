# CLI Commands

In this sub-section of the cli section, we will cover all the commands that are available to the CLI.

## Commands
- `init` - Initialize a new project
- `run <command>` - Run a command specified in the contenthook config
- `push` - Push the content of your content files (MD, MDX, MARKDOWN) to the ContentHook Cloud
- `pull` - Pull the content from the ContentHook Cloud to your local project
- `help` - Shows a help menu for the CLI
- `version` - Shows the version of the CLI
